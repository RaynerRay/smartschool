| `\`                      | `&#92;`     | Backslash             |
| `©`                      | `&copy;`    | Copyright symbol      |
| `®`                      | `&reg;`     | Registered trademark  |
| `™`                      | `&#8482;`   | Trademark symbol      |
| ` ` (non-breaking space) | `&nbsp;`    | Non-breaking space    |
| `¢`                      | `&cent;`    | Cent sign             |
| `£`                      | `&pound;`   | Pound sign            |
| `¥`                      | `&yen;`     | Yen sign              |
| `€`                      | `&euro;`    | Euro sign             |
| `§`                      | `&sect;`    | Section symbol        |
| `¶`                      | `&para;`    | Paragraph symbol      |
